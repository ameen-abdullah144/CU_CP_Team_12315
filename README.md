# 🚀 Smart AI Surveillance & Alert System

An AI-based smart surveillance system that uses YOLO for real-time object detection from CCTV feeds. The system provides a centralized dashboard, supports multiple camera streams, and generates automated alerts for enhanced security and monitoring.

---

## 📌 Features

- 🔍 Real-time object detection using YOLOv4-tiny  
- 📹 CCTV/IP camera integration (RTSP support)  
- 📊 Centralized dashboard for monitoring multiple cameras  
- ⚡ Automated alert system for suspicious activities  
- 🧠 Reduces manual monitoring using AI  
- 🌐 Works for traffic surveillance and home security  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Libraries:** OpenCV, NumPy  
- **Model:** YOLOv4-tiny  
- **Tools:** VS Code, Git, GitHub
